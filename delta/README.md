# DELTA

This is a core package of **DELTA** Deep Learning library as well as `rust` bindings.

## Setup

This project requires the version of `Rust` >= 1.95.

To build the library run:

```sh
cargo build
```

Ensure that no test fails (in project root will work as well):

```sh
cargo test
```

## Usage

See `/examples` for usage examples. To run an example, execute the following line passing a file name:

```sh
cargo run --example <name>
```
